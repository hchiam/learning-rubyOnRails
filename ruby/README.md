# To run in interactive mode in terminal:

In Terminal/Commandline type (and then hit enter):

    irb

(Then run some code.)

Then to quit, just type (and then hit enter):

    quit

# To run a ruby file (.rb) using terminal command:

In Terminal/Commandline type (and then hit enter):

    ruby filename.rb
